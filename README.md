# React Todo App

A simple Todo application built with React.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Task counter
- Component-based structure

## Tech Stack

- React
- JavaScript
- CSS
- Vite

## What I Learned

- React useState hook
- Passing props between components
- Handling form input
- CRUD operations in React
- Component refactoring
- Git & GitHub workflow

## How to Run Locally

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
